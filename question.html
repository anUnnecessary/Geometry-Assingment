What is DOM and What is the purpose of DOM?

DOM stands for Document Object Model and it can be represented as a tree of elements created by the browser. With DOM manipulation, you can create, modify, style, or delete elements without a refresh. It also promotes user interactivity with browsers. DOM manipulation is the interaction of the JavaScript DOM API to modify or change the HTML document. All Ul libraries and frameworks are built based on this DOM Manipulation, Like---

  • jQuery
  • Angular
  • React
  • Vue.js

    • What is a callback function and Why will you use it?

    A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

    • 1-When working with the file system (Downloading or Uploading).
    • 2-Sending the network request to get some resources such as test or binary file from the server.
    • 3-Events.
    • 4-The DOM in the browser.
    • 5-Working with web APIs to fetch data.

    What is event bubble?

    The Event Bubble is similar to the gas developed underwater and it slowly floats to the top and we see it as a bubble. When we click something on a website and it does something, we call this an event bubble. For example, pressing the like button on Facebook shows as a like.
